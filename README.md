# UBC MDS Leetcode sessions

This repository contains solutions for LeetCode problems practiced during the UBC Master of Data Science weekly coding sessions. Each folder corresponds to a single LeetCode problem and contains solution files. When contributing your own solutions, create a new file with your name in the relevant problem folder to maintain clear authorship.

---

## Problems Covered

- TwoSum  
- ValidAnagram  
- ValidSudoku  
- ValidPalindrome  
- StringCompression  

Additional problems and solutions may be added as the sessions progress.

## How to Use

1. Clone this repository.  
2. Navigate to the folder of the problem you want to study or practice.  
3. Review existing solutions or add your own solution file named after you (e.g., `yourname.py`).  
4. Test and refine your approach locally.
